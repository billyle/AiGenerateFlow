# 提示词调试面板功能增强

## 功能说明

本次更新为插件添加了可折叠的提示词调试面板，方便用户在需要时查看和修改AI生成流程图使用的提示词模板。

## 界面布局

采用了三层嵌套分割面板结构：
1. **主水平分割面板**：左侧（70%）vs 右侧图形面板（30%）
2. **左侧垂直分割面板**：上方面板（30%）提示词调试 vs 下方面板（70%）代码展示
3. **提示词面板**：默认折叠，可展开编辑

## 主要特性

### 可折叠设计
- 默认状态下提示词面板处于折叠状态，节省界面空间
- 点击标题栏的 ▶/▼ 按钮可以展开/折叠面板
- 标题会相应显示当前状态

### 调试功能
- 显示当前使用的提示词模板
- 支持实时编辑提示词内容
- 提供"重置为默认"按钮恢复原始模板
- 提供"应用修改"按钮确认更改

### 智能应用
- 修改后的提示词在下次生成时自动生效
- 系统会检测提示词是否被修改
- 未修改时继续使用默认模板

## 技术实现

### 组件结构
- 使用 `JSplitPane` 实现可调整的分割面板
- 采用 `BorderLayout` 管理面板布局
- 通过数组引用实现组件间通信

### 状态管理
- 面板展开/折叠状态的切换
- 文本内容的保存和恢复
- UI组件的动态更新

## 使用场景

1. **日常使用**：面板默认折叠，不影响正常使用流程
2. **调试优化**：需要调整提示词效果时展开面板进行修改
3. **模板定制**：根据不同项目需求定制提示词模板
4. **效果对比**：快速测试不同提示词的效果差异

## 注意事项

- 提示词修改仅在当前会话中生效
- 建议在修改前备份重要的提示词模板
- 过度复杂的提示词可能影响生成效果
- 修改后需要重新生成才能看到效果
